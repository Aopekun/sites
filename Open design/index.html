Локальный запуск через Docker

Это безопасный путь по умолчанию на нашем сервере, потому что репозиторий требует Node 24.

cd ~/.openclaw/workspace-tech/open-design/deploy
cp .env.example .env
openssl rand -hex 32
# вставить токен в OD_API_TOKEN= внутри .env
docker compose up -d
# открыть: http://localhost:7456

Важно: токены не присылать в чат. Их надо хранить только в .env/config.
